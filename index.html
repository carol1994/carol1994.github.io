用hexo搭建博客时遇到的问题

1.配置_config.yml，冒号和后面的内容之间要留有空格

2.必须先在github上面创建仓库，以账号名为仓库名的一部分，例如：carol1994.github.io

3.创建仓库的同时要初始化，并且设置好ssh key

4.用github的http协议会使得部署不成功（未解决）

5.部署的时候（hexo d）要在git环境下进行，否则会permission denied

如何用hexo搭建博客

1.安装 npm install -g hexo

2.初始化 hexo init hexo-demo

3.跳转目录 cd hexo-demo

4.安装依赖 npm install

5.启动服务器 hexo server

6.浏览器登录 http://localhost:4000/
