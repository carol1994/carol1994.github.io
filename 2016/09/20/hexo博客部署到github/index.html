hexo博客部署到github

1.新建文件 hexo n 文章名

2.到_posts文件中找到该文件进行编写（记得修改编码）

2.5 github上要创建相关项目，例如carol1994.github.io，并且初始化

3.修改_config.yml文件，冒号和后面的内容要空格

deploy:
type: git
repo: git@github.com:carol1994/carol1994.github.io.git
branch: master

同时也修改site，url等

repo使用的是ssh，使用前需要配置

$ cd ~/. ssh 检查本机的ssh密钥

如果提示：No such file or directory 说明你是第一次使用git

生成新的密匙：$ ssh-keygen -t rsa -C “邮件地址@youremail.com”

然后系统会要你输入密码：Enter passphrase (empty for no passphrase):<输入加密串> Enter same passphrase again:<再次输入加密串>

然后复制id_rsa.pub文件中的所有内容粘贴到github账户的new ssh key

$ ssh -T git@github.com验证是否成功设置ssh

4.生成本地静态文件 hexo g

5.安装 npm install hexo-deployer-git -S(第一次)

6.hexo d

7.浏览器中打开http://carol1994.github.io
